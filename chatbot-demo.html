Embeddable Chat Widget

꽃배달 상담 챗봇 설치 데모

이 페이지에는 아래 한 줄만 들어가 있습니다. 우하단 “상담하기” 버튼을 눌러보세요. 우측에서 챗봇이 슬라이드로 등장합니다.

1. 설치 — 이 한 줄이면 끝

<script src="https://YOUR-HOST/assets/chatbot/embed.js" defer></script>

· YOUR-HOST를 이 가이드가 호스팅된 도메인(예: GitHub Pages 주소)으로 바꾸기만 하면 됩니다.
· 챗봇 UI(chatbot.html)와 데이터 경로는 스크립트가 자기 위치에서 자동으로 찾습니다. 다른 사이트에 붙여도 그대로 동작합니다.

2. 옵션 — data-* 속성으로 커스터마이즈

<script src="https://YOUR-HOST/assets/chatbot/embed.js"
        data-label="꽃 상담하기"
        data-accent="#e60023"
        data-position="right" defer></script>
data-title헤더 제목 (기본: 꽃배달 상담 도우미)
data-subtitle헤더 부제
data-label버튼 글자 (기본: 상담하기)
data-accent포인트 색 hex (기본: #e60023)
data-positionleft / right (기본: right)
data-open"true"면 처음부터 열림
data-api(선택) LLM 백엔드 엔드포인트 — 없으면 내장 검색 엔진으로 답함

3. 직접 제어

전역 window.FlowerChat 로 열고 닫을 수 있습니다.

<button onclick="FlowerChat.open()">지금 상담하기</button>

· FlowerChat.open() / .close() / .toggle()
· 답변은 이 가이드의 FAQ·전 챕터 내용에서 자동으로 찾아옵니다. 가이드를 업데이트하면 챗봇 답변도 함께 최신화됩니다.
